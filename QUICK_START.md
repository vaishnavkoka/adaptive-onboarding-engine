# 🚀 Quick Start Guide - AI-Adaptive Onboarding Engine\n\n## ⚡ 5-Minute Setup\n\n### Step 1: Set Up Python Environment\n```bash\ncd \"AI-Adaptive Onboarding Engine\"\npython -m venv venv\nsource venv/bin/activate  # Windows: venv\\Scripts\\activate\npip install -r requirements.txt\n```\n\n### Step 2: (Optional) Set Up LLM\n```bash\n# Install Ollama from https://ollama.ai\n# In a separate terminal, start Ollama:\nollama pull deepseek-r1:7b\nollama serve\n```\n\n### Step 3: Run Application\n```bash\npython app.py\n# Opens at http://localhost:3000\n```\n\n### Step 4: Use Application\n1. **Upload Resume** (PDF, DOCX, or TXT)\n2. **Upload Job Description** (text or file)\n3. **Choose Category** (24 options available)\n4. **Select Mode** (Fast keyword or LLM for better accuracy)\n5. **Click \"Analyze Skills\"**\n6. **View Results** with score breakdown and learning path\n7. **Export as CSV** or copy pathway\n\n---\n\n## 📊 What You'll See\n\n### Match Score Card\n- **0-100%** score showing resume-to-job match\n\n### Score Breakdown (NEW!)\n- **Your Skills**: Skills found in resume\n- **Required Skills**: Skills needed for job\n- **Matching Skills**: How many match\n- **Proficiency Matches**: Bonus-eligible matches\n- **Base %**: (Matches / Required) × 100\n- **Proficiency Bonus**: +3 per match\n- **Final Score**: Calculation result (max 100)\n\n### Score Interpretation (NEW!)\n- **🟢 80-100**: Excellent Match - Minor learning needed\n- **🟡 60-79**: Strong Match - Some development\n- **🟠 40-59**: Moderate Match - Structured learning\n- **🔴 20-39**: Entry-Level - Comprehensive training\n- **⚠️ <20**: Early Career - Dedicated learning\n\n### Skills Gap Card\n- Missing skills from resume\n- Proficiency gaps (resume < job level)\n\n### Learning Pathway Card\n- **Risk-ranked** tasks (most important first)\n- Estimated hours & difficulty\n- Prerequisites-aware sequencing\n\n### Reasoning Trace\n- How score was calculated\n- Decision logic for recommendations\n- Full transparency\n\n---\n\n## 🎛️ Extraction Modes\n\n### Fast Mode (Default)\n- **Speed**: ~5 milliseconds\n- **Accuracy**: 63.4% average\n- **No Dependencies**: Works offline\n- **Use When**: Quick results needed, LLM unavailable\n\n### LLM Mode (Optional)\n- **Speed**: 10-15 seconds typical (30s timeout)\n- **Accuracy**: 78.2% with domain terms\n- **Benefits**: Context-aware, rare skills, implicit skills\n- **Requires**: Ollama + DeepSeek-R1 7B running\n- **Use When**: Maximum accuracy needed\n\n**Automatic Fallback**: If LLM times out, reverts to Fast mode\n\n---\n\n## 🔧 Troubleshooting\n\n### Port Already in Use\n```bash\n# Find what's using port 3000\nlsof -i :3000  # Mac/Linux\nnetstat -ano | findstr :3000  # Windows\n\n# Kill the process or change port in app.py\n```\n\n### \"Ollama not found\" Message\n- **If using Fast mode**: No problem, works without Ollama\n- **If using LLM mode**: Start Ollama server:\n  ```bash\n  ollama serve\n  ```\n- Or install Ollama from https://ollama.ai\n\n### Timeout on LLM Analysis\n- Fallback to Fast mode is automatic\n- If frequent, your system may be slower:\n  1. Reduce system load\n  2. Use Fast mode instead\n  3. Check Ollama is running: `curl http://localhost:11434`\n\n### File Upload Issues\n- **PDF**: Install PyPDF2 (included in requirements.txt)\n- **DOCX**: Install python-docx (included)\n- **Large Files**: Client-side upload limit ~50MB\n\n---\n\n## 📁 Project Structure\n\n```\nAI-Adaptive Onboarding Engine/\n├── app.py                      # Main Flask application (port 3000)\n├── skill_extractor.py          # Dual-mode skill extraction\n├── onboarding_engine.py        # Fair scoring & analysis engine\n├── adaptive_pathway.py         # Learning pathway generation\n├── requirements.txt            # Python dependencies\n├── templates/\n│   └── index.html              # Web interface\n├── static/\n│   └── style.css               # Styling\n├── diagrams/                   # 4 presentation diagrams\n│   ├── 1_system_architecture.mmd\n│   ├── 2_data_flow.mmd\n│   ├── 3_ui_ux_logic.mmd\n│   └── 4_scoring_architecture.mmd\n├── README.md                   # Project overview\n├── DIAGRAMS_GUIDE.md           # Diagram documentation\n├── IMPLEMENTATION_COMPLETE.md  # Complete status report\n└── QUICK_START.md              # This file\n```\n\n---\n\n## 🎯 Example Workflow\n\n### Step-by-Step Example\n\n1. **Prepare Files**\n   -
